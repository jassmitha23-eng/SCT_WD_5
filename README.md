# 📝 Advanced To-Do Web App

A modern and interactive To-Do Web Application built using HTML, CSS, and JavaScript. This app allows users to efficiently manage tasks with features like adding, editing, deleting, marking completion, and setting date & time.

---

## 🚀 Features

* ➕ Add new tasks
* ✏️ Edit existing tasks
* ✅ Mark tasks as completed
* ❌ Delete tasks
* 🕒 Set date & time for tasks
* 💾 Data stored using Local Storage
* 🎨 Modern Glassmorphism UI
* 📱 Fully responsive design

---

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript

## ⚙️ How to Use

1. Open the application in your browser
2. Enter a task in the input field
3. Select date & time (optional)
4. Click **Add Task**
5. Use ✔ to mark complete
6. Use ✏️ to edit tasks
7. Use ✖ to delete tasks
